Timeboxed watchface

Pebble Health
Enables showing Pebble Health data. We show steps and distance by default, but you can enable showing sleep and deep sleep times after you wake up. (Not available for classic Pebble).
If set, the watch will show the sleep and deep sleep time for that night for an half hour after you wake up, switching back to steps after that time.
Weather
Enables showing weather data. It uses OpenWeatherMap data by default, which is less accurate. For better accuracy, use WeatherUnderground. This requires a WeatherUnderground API key. Go to wunderground.com to create a free account and get a key and insert it below.
WeatherUnderground Key
Manual location
If you define a manual location, we won't try to use your current location for weather info. (max. length 64 characters)
Notice: If you're not sure the city you entered is working, use the "Verify" button to check if it's a valid location, as sometimes you need to fiddle a little bit with the location string to get the correct result, e.g., adding or removing the state.
Bluetooth
Emits a vibration notification when the watch is disconnected from the phone. Won't trigger if you're asleep (only on Time and Time Round).
Updates
Checks daily if there's a new version of the watchface available. If there's an update, you'll see a little arrow on the left side of the screen. To update, go to the Pebble app -> My Pebble -> Watchfaces, tap on the Timebox watchface and select 'About this app'.
